# Additional Reading

The following papers explore advanced topics that build on the ideas introduced in this workshop, including programmable data planes and network monitoring for modern 5G and transport networks.

N. Saha, M. T. Arashloo, N. Shahriar, R. Boutaba.  
**Dynamic SLA-aware Network Slice Monitoring.**  
ACM SIGMETRICS '26, Ann Arbor, Michigan, USA, June 8–12, 2026.  
[PDF](https://rboutaba.cs.uwaterloo.ca/Papers/2026/saha-sigmetrics26.pdf)

This work explores how monitoring systems can dynamically adapt telemetry collection to satisfy service-level objectives in network slices, enabling scalable and efficient observability in modern programmable networks.

M. Rouili, R. Boutaba.  
**Blink: A P4-Based 5G Centralized Unit.**  
IEEE/IFIP Network Operations and Management Symposium (NOMS), Honolulu, HI, USA, May 12–16, 2025.  
[PDF](https://rboutaba.cs.uwaterloo.ca/Papers/Conferences/2023/rouili-blink-noms25.pdf)

This paper demonstrates how programmable switches using the P4 language can offload 5G user-plane processing from servers to high-speed network hardware, significantly improving packet processing performance and latency.
