# Learning Outcomes

After completing this workshop, participants will be able to:

- Understand the role of **Software-Defined Networking (SDN)** in transport networks
- Inspect and modify **data-plane forwarding behavior** using Open vSwitch
- Understand how **SDN controllers manage network state**
- Use the **ONOS controller** to establish network connectivity
- Observe how controllers dynamically program forwarding paths
