# Learning Outcomes

By the end of this workshop, participants will:

- Understand how modern transport networks can be programmatically controlled rather than configured manually on individual devices.

- Understand the separation between data plane and control plane, and how centralized control enables more flexible network management.

- Gain hands-on experience with programmable forwarding behavior, observing how packet forwarding decisions can be dynamically modified in a network.

- Observe how a centralized controller can maintain a global view of the network, compute paths, and install forwarding rules automatically.

- Understand how transport networks can enforce service differentiation under congestion, enabling certain traffic classes to maintain better performance when resources are limited.
